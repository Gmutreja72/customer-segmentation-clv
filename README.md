# Retail Customer Segmentation & CLV Analysis
**Tools:** SQL · Excel · Looker Studio
**Date:** August 2024

## Overview
Segmented 10,000+ retail customers using RFM (Recency, Frequency, Monetary) analysis
to identify high-value customers and improve marketing ROI.

## Methodology
1. Extracted and cleaned transactional data using SQL
2. Scored customers on Recency, Frequency, and Monetary dimensions
3. Created 4 segments: Champions, Loyal, At-Risk, Lost
4. Calculated Customer Lifetime Value (CLV) for each segment

## Key Results
- Top 20% of customers contribute 65% of total revenue
- At-Risk segment: 2,300 customers with recovery potential
- Projected 22% improvement in campaign ROI through targeted outreach
